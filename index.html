Philosophy

Technology

Layered Architecture

Enterprise layered design with dependency injection across both platforms. API routes, service layers, repository layers, and external adapters — all connected through typed interfaces. Swapping a provider means changing one line in the composition root.

Open Source

MIT licensed tools built in the open. 300+ commits and 60+ merged PRs across both platforms. Community-driven development with transparent roadmaps.

Music-First

Built by musicians, for musicians and music culture. Every feature informed by real needs from the underground electronic music scene — from booking logistics to flier archival.

Adapter Pattern

9 abstract interfaces in raiveFlier, 22 repository interfaces in Resistor. LLMs, OCR engines, music databases, payment providers, vector stores — all accessed through contracts. No vendor lock-in, ever.

Dual Stack

Python 3.12 with FastAPI and Pydantic v2 for raiveFlier's research engine. TypeScript with Next.js and Prisma ORM for Resistor's booking marketplace. PostgreSQL, ChromaDB, WebSocket, and PayPal Commerce across the ecosystem.

Security

OWASP Top-10 scanning, dependency auditing, secret detection, TLS enforcement. Zod validation on every API boundary, custom error hierarchies, and structured JSON logging across all services.
