Description

Single-page static website for Resistor Technology, a company building tools for underground music culture. Showcases two products: raiveFlier (AI rave flier research tool) and Resistor (self-booking tour platform for musicians). Features an interactive synthesizer-styled hero section with draggable knobs, faders, and a scroll-driven VU meter.

Setup / Quick Start

Prerequisites

  • Any modern web browser (Chrome, Firefox, Safari, Edge)
  • No build tools, Node.js, or package managers required

Run Locally

Open index.html directly in a browser, or serve with any static server:

cd /Users/aaandy/_andy_ai_projects_2026/resistor-technology-site
python3 -m http.server 8080

Then open http://localhost:8080

Design System

The "Bunker + Brass" palette draws from the shared design language of both products:

  • Brass (#c4a67a) — Resistor product accent, primary brand color
  • Amethyst (#6a4a8a) — raiveFlier product accent
  • Teal (#4a8a7a) — Technology/trust sections
  • Bunker (#0a0a0c) — Dark base palette with warm undertones

Typography: Space Grotesk (display), Inter (body), IBM Plex Mono (technical/labels).

Interactive Features

  • GRAIN knob — Controls film grain overlay opacity across the page
  • GLOW knob — Controls accent color glow intensity on interactive elements
  • DEPTH fader — Controls parallax depth for scroll animations
  • VU meter — Needle responds to scroll velocity in real time
  • Section LEDs — Light up as corresponding sections enter the viewport
  • OUTPUT patch jack — Smooth-scrolls to contact section

Accessibility

  • WCAG 2.1 AA compliant color contrast (parchment on bunker-900 ≈ 12:1 ratio)
  • Semantic HTML5 landmarks (<header>, <main>, <section>, <footer>)
  • All interactive elements have aria-label, role="slider" or role="button"
  • Keyboard navigation for knobs, faders, and patch jack (Arrow keys, Tab, Enter)
  • Skip-to-content link
  • prefers-reduced-motion fully supported — all animations disabled
  • Focus-visible rings on all interactive elements

Deployment

Deployed as a Render static site:

  1. Build command: echo "No build needed"
  2. Publish path: . (root)
  3. Custom domain: resistor.technology
  4. HTTPS enforced automatically
